> Commands available:
add: adds a new resource to the library inventory.(e.g. add /i ISBN /t TITLE /a AUTHOR /tag TAG [/g GENRE /s STATUS])

eventdelete: Delete an event in the event list based on the index (e.g. eventdelete /i INDEX)
